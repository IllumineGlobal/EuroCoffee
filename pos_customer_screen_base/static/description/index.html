Odoo POS Customer Screen Base

The base module provides the necessary structure to support the features to run POS customer Screen in Odoo.



You would need to install Odoo POS customer Screen base module first for its dependencies to work properly.

Note : The module serves as the base for the Odoo POS Customer Screen Modules:

  • POS Customer Review Screen
  • POS Customer Cart Screen

Pos Customer Review Screen

  • The module adds a customer Facing Feedback Screen to the Odoo POS.
  • It works without the need of any physical connection with Odoo POS. Works with or Without POS BOX.
  • You can take customer feedback using the screen while processing their orders.
  • Allows you to record customer ratings for each POS Shop and thus POS users.
  • Project the customer feedback screen on any display.


Pos Customer Cart Screen

  • The module adds a customer Facing Cart View Screen to the Odoo POS.
  • The customer can see the details of his Odoo POS cart as your build his cart.
  • It works without the need of any physical connection with Odoo POS. Works with or Without POS BOX.
  • The customer Screen shows the cart details such as product added to the cart, order total amount paid, etc.
